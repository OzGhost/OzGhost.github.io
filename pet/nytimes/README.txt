
Environment requirement:
  - NodeJS
  - npm

Installation
  - Extract file named 'nytimes.zip' somewhere
  - Open terminal/cmd and change working directory to the place
    where the file was extracted
  - Run the command below:
      npm install

Some command:
  npm run build         : transform jsx into js
  npm run watch         : monitor related jsx files and re-transform
                          every time these files has changed
  npm run style-build   : transform sass into css
  npm run style-watch   : monitor specified sass file and re-compile
                          when this file has changed
